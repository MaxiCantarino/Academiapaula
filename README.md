# Academia Paula - Plataforma Educativa

Proyecto web para la promoción y dictado de cursos, servicios estéticos y portal de alumnos.

## 🛠 Tecnologías

- **Lenguajes:** HTML5, CSS3 (con Flexbox/Grid), JavaScript Vanilla
- **Backend y Base de Datos:** Supabase (Auth & Database)

## 🚀 Guía de instalación rápida

1. **Clonar el repositorio:**
   ```bash
   git clone 
   cd Academiapaula
   ```

2. **Configurar las variables de entorno:**
   Copia el archivo de ejemplo `.env.example` y renómbralo a `.env`. Completa las variables correspondientes proporcionadas por Supabase:
   ```bash
   cp .env.example .env
   ```

3. **Ejecutar en desarrollo:**
   Al ser un proyecto Vanilla de HTML/CSS/JS, basta con usar una extensión como `Live Server` en VS Code o servir los archivos mediante cualquier servidor simple de desarrollo en el directorio raíz.
